# Orchid Night Club - Next.js Project

This is a Next.js 14 project for Orchid Night Club website, built with Tailwind CSS and JSX.

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Project Structure

- `app/` - Next.js 14 App Router files
  - `layout.js` - Root layout
  - `page.js` - Home page
  - `globals.css` - Global styles with Tailwind
- `components/` - React components
  - `Header.js` - Navigation header
  - `Hero.js` - Hero section with logo
  - `Packages.js` - Package pricing section
  - `FAQ.js` - Frequently asked questions
  - `Footer.js` - Footer with contact info

## Technologies Used

- Next.js 14
- React 18
- Tailwind CSS
- JSX (No TypeScript)

## Features

- Responsive design
- Dark theme
- Bengali and English content
- Package information display
- FAQ section
- Contact information
